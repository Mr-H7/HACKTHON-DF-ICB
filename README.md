# HackSphere 2026

مسابقة الهاكاثون السنوية بالتعاون بين مدرسة ICB وفريق Digital Fingers

## المميزات
- تصميم عصري وجذاب
- دعم الوضع المظلم والفاتح
- دعم اللغتين العربية والإنجليزية
- لوحة تحكم للفرق
- لوحة تحكم للمديرين
- نظام تسجيل متكامل
- لوحة المتصدرين

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome
- Google Fonts

## التشغيل المحلي
1. قم بتنزيل الملفات
2. افتح index.html في المتصفح

1. ارفع الملفات إلى GitHub
